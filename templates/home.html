Mini-Hackathon: Predicting Online Shoppers' Purchasing Intention

Overview

Welcome to the Data Science Mini-Hackathon, organized by PWSkills, a leader in empowering aspiring data scientists with practical, hands-on learning experiences. Open to all enthusiasts, from beginners to seasoned practitioners, this bi-monthly Hackathon fosters innovation and technical expertise in data-driven solutions. This edition, themed around the Predicting Online Shoppers' Purchasing Intention challenge, invites participants to tackle a real-world problem in e-commerce analytics using machine learning. By predicting whether an online shopping session will end in a purchase based on visitor behavior and session attributes, participants will contribute to strategies that enhance conversion rates, reduce cart abandonment, and optimize user experiences in virtual shopping environments while honing their data science skills.

Problem Statement

The objective of this Hackathon is to develop a machine learning model that predicts whether an online shopping session will result in a purchase (the target variable, “Revenue”), based on a combination of behavioral, temporal, and session-related features. This is a binary classification problem, where the goal is to maximize prediction accuracy while addressing class imbalance, as most sessions do not end in purchases. The challenge leverages the Online Shoppers Purchasing Intention Dataset, enabling participants to explore data-driven insights that can improve e-commerce personalization, such as recommending content to high-intent visitors or intervening in sessions at risk of abandonment. Participants will analyze features such as page visits by category, bounce rates, exit rates, and session duration to simulate real-world decision-making in e-commerce optimization.

Dataset Overview

The Online Shoppers Purchasing Intention Dataset is a curated collection of session-level attributes capturing visitor behavior on an e-commerce website, designed to predict whether a session culminates in a purchase. Derived from real-world e-commerce interactions over a one-year period, this dataset includes 12,330 unique sessions from different users, ensuring diversity and avoiding biases from repeated user behaviors or seasonal campaigns. It emphasizes the integration of clickstream data (e.g., page visits and durations) with session metadata (e.g., browser, region) to model purchasing intent, making it an excellent resource for exploring machine learning applications in e-commerce. Each row represents a complete session, labeled with whether it ended in a revenue-generating transaction, providing a foundation for binary classification tasks that mirror practical challenges in online retail analytics.

Special Day Float Closeness to special days (e.g., Valentine's Day, 0-1 scale, peaking at 1 near the event).

Browser Categorical Visitor's browser (e.g., Chrome, Firefox).
Region Categorical Geographic region of the session start.
TrafficType Categorical Source of traffic (e.g., banner, SMS, direct).
VisitorType Categorical Type of visitor (New Visitor, Returning Visitor, Other).
Weekend Boolean Whether the visit occurred on a weekend (True/False).
Month Categorical Month of the visit (e.g., Jan, Feb).
Revenue Boolean Target variable indicating if the session ended in a purchase (True/False, imbalanced with ~84.5% False).

Task: Primarily, predict the "Revenue" label (binary classification: purchase or no purchase). Optionally, for advanced exploration, predict abandonment likelihood using sequential clickstream data (e.g., page types and durations over time). Challenges include handling categorical variables, addressing class imbalances, and ensuring robust feature engineering.
